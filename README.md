## 本地预览
1. 安装 `node.js/npm`
> 可以参考我的文章 ➡️ [Win下安装node和npm]()

2. 运行
```bash
npm install
npm run dev
```

3. 生成

```bash
npm run build   //或者 yarn run build
```

## 项目支持

### [@korey/MVVM](https://github.com/zhaoky/mvvm)

> 一个简易的 MVVM 框架，目前实现了 data-binding 和 view-refresh 的功能，仍在不断优化更新，欢迎学习交流。
